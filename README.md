Hello!
My name is liam and this is my readme for kal-kool-ator!
My project is a all around toolkit for nerdiness, this contains things such as pi calculators, ctfs, conversions, music players and a lot more!
I used a variety of stacks including sfml, cmath and just regular c++ imports like stdlib and iostream.
First, I thought of what I wanted to make in my head, wrote some notes about the layout and how it would run and got to working, I built all of the simple stuff first like addidion and subtraction, but then I moved to the brain melting stuff like the pi calculator(more about that later), and the repetitive stuff which was the converter, was there a simpler way to do it, probably, do I have the brain power to do so, hell no. 
for the music player, it was not playing any of the songs so I needed to do a good couple of hours of debugging but I realised I had the "music.play" function in the while loop which meant it was playing it so fast every second the computer couldn't process it, but I eventually figured that out, I am only at the 20 hour mark so there is more features to come, like not blowing up the computer when you run the pi calculator for too long or adding more features like a ascii converter and hopefully much more if i do not get rejected(please note most of the things like audio and images are placeholders, once i complete the 35 hours i will do actual covers of songs, this is just to get a sensus of what i should add and make sure it is ready):)

## how to use: 
1. go to my github @ [mon github](https://github.com/lcarpello/horizons)
2. download appimage from the release tab at the right under kal-kool-ator
3. run chmod +x kalkoolator-x86_64.AppImage
4. run the appimage with ./kalkoolator-x86_64.AppImage

I tested this on arch linux with the command ./kalkoolator-x86_64.AppImage
I do not believe you need any other dependencies, even so they shouldn't be too hard to access.
